### Code example:

This function iteratively finds the smallest divisor of a given number:

```javascript
const smallestDivisor = num => {
  if (num === 1) {
    return 1;
  }
  let divisor = 2;
  while (num % divisor !== 0) {
    divisor++;
  }
  return divisor;
}
```
